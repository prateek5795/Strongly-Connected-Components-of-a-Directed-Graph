Navigation Inside the Program
- Enter values in the following format:  
n m    v1 v2 w    v1 v2 w    0  
where,  
n: number of vertices,  
m: number of edges,  
v1: source vertex,  
v2: destination vertex,  
w: weight of edge and  
0: marks the end of input.
